# Zamanai Explorer 📂
This is a custom Android File Explorer app built using **Python** and **Kivy**.

## Features
* **Zamanai Red Theme:** A custom UI designed for a modern look.
* **File Management:** Browse files and folders on your Android device.
* **Built with GitHub Actions:** The APK is automatically compiled using Buildozer in the cloud.

## How to get the APK
1. Go to the **Actions** tab in this repository.
2. Click on the most recent successful build.
3. Download the file from the **Artifacts** section at the bottom.
